## What this app does:

It listens to a keypress (By default double click F4)

It sends the screenshot along with extracted text to gpt-04

The reply will change the color (RGB) of your selected device (Can be MOBO, Keyboard, Mouse)

This is not intended for use. It is only built for fun and as a submission for Mirothon


## Requirements:
1. Tesseract
2. OpenRGB
3. See Requirements.txt
